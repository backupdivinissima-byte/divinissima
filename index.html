Dados em tempo real entre computadores

Quando ativada, toda vez que você importar um arquivo ou alterar dados, eles são enviados automaticamente para uma planilha do Google. Os vendedores, em qualquer computador, recebem essas atualizações a cada 15 segundos.

Cole abaixo a URL do seu Google Apps Script Web App. Não sabe como criar? Ver guia passo a passo

📖 Como criar o Web App no Google
  1. Acesse sheets.google.com e crie uma planilha nova (ex: "Divinissima - Dados")
  2. No menu, clique em Extensões → Apps Script
  3. Apague o código padrão e cole o script fornecido (peça para o suporte ou veja o documento anexo)
  4. Clique em Implantar → Nova implantação
  5. Tipo: App da Web · Executar como: Eu · Quem pode acessar: Qualquer pessoa
  6. Clique em Implantar e autorize o acesso quando solicitado
  7. Copie a URL do app da Web gerada (termina em /exec)
  8. Cole essa URL no campo acima e clique em Salvar e ativar
